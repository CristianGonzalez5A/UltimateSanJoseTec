BlueGriffon es un editor de HTML de código abierto y visual basado en el motor de renderizado Gecko, el mismo que utiliza el navegador Mozilla Firefox. Su propósito es proporcionar una plataforma de edición web potente y accesible tanto para desarrolladores web experimentados como para aquellos sin experiencia previa en programación. BlueGriffon permite crear sitios web de manera visual e interactiva, sin la necesidad de escribir mucho código a mano, aunque también ofrece un editor de código para aquellos que prefieren tener un control más profundo sobre el diseño y la estructura del sitio web.
